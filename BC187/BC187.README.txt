S. cerevisiae strain BC187 genome sequence.

02/03/2015

This directory contains 6 files describing the genome of S. cerevisiae strain 
BC187, a spore derivative of UCD2120 which was isolated from a wine barrel 
fermentation in California's Napa Valley (Accession: JRII00000000). 

genome fasta file: BC187_Stanford_2014_JRII00000000.fsa
nucleic acid coding sequence file: BC187_JRII00000000_cds.fsa
protein sequence file: BC187_JRII00000000_pep.fsa
GFF file: BC187_JRII00000000.gff
snv variant call format file: BC187.gatk.vcf.gz
indel variant call format file: BC187.indel.gatk.vcf.gz

FASTA headers in each of the files include the following pieces of information:

genome fasta file: 
 GenBank GI number
 GenBank accession.version
 species name
 strain name
 scaffold number
 contig length

nucleic acid coding sequence file: 
 ORF name_strain name
 gene name
 feature type
 feature classification (Verified, Uncharacterized, or Dubious)
 feature description

protein sequence file: 
 ORF name_strain name
 gene name
 feature type
 feature classification (Verified, Uncharacterized, or Dubious)
 feature description

Feature definitions (for coding sequences and features in the gff file) were
assigned via homology-based annotation using LASTZ and AUGUSTUS vs. the R64-1
S288C reference sequence in conjuction with ab initio annotation using MAKER 
and BLASTX validation.
